## Installation
You can use any of your favorite vim plugin manager to install or drop `galaxy.vim` into `~/.vim/colors/`

Afterwards, add the following into your `vimrc` if not already added:
```vim
syntax on
set t_Co=256
set background=dark
colorscheme galaxy
```
